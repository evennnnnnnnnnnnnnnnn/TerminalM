# Welcome to your Flint!

This document is the entry point to this workspace.

## About This Flint

**TerminalM** is a modern Electron-based terminal emulator with tabs, split panes, search, a Catppuccin theme, and a 3D virtual office playground. The codebase lives at the parent directory (`../`) relative to this Flint root.

## Key Paths

- **Source code**: `../` (main.js, src/main/, src/renderer/, src/preload/)
- **Code Docs**: `Mesh/Code Docs/TerminalM/`
