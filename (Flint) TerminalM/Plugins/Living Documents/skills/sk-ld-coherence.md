# Skill: Full Sync Living Documents

Update all living documents to ensure coherence.

# Actions

1. Find all `#ld/living` documents
2. Read all living documents that have been modified since last commit
3. Make edits to synchronize everything
4. Report what was updated and why

# Constraints

- Only modify `#ld/living` documents
- Never modify `#ld/dead` documents
- Preserve document voice and structure
- When in doubt, don't update
