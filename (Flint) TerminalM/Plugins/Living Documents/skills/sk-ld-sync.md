# Skill: Sync Living Documents

Update living documents that have drifted from reality.

# Actions

1. Find all `#ld/living` documents
2. For each, check if it references anything changed this session
3. Make edits to synchronize
4. Report what was updated and why

# Constraints

- Only modify `#ld/living` documents
- Never modify `#ld/dead` documents
- Preserve document voice and structure
- When in doubt, don't update
