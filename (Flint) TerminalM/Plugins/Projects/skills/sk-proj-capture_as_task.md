# Skill: Capture As Work

Capture the latest actions done in the session as a task

# Actions

- Create a task using the @tmp-proj-task template which captures the work done recently. Make sure to find the correct latest task number so the new task is numbered correctly. 
- Becuase the task has already been done, make sure the task status should be marked as done
- Make sure the task is referenced by the right documents, for example, if it was done within an increment, make sure the task is appended to the increment's log. 
- Report any actions to the user.
